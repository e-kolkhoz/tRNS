TP4054S5 - контроллер заряда

PLUS_BAT_ADC // аналоговый вход с делителя 360k/100k батарейки 3.7V 
VBUS_STAT // вход детектор VBUS c USB c делителя 51k/100k относительно входа CHRG TP4054S5 надо делать pinMode(VBUS_STAT, INPUT_PULLDOWN); 
EN_WAKEUP // выход вкл. аналоговые модули
USB_DET  // вход детектор VBUS c USB c делителя 51k/100k

надо уметь из меню загонять устройство в энергосберегающий режим и 
сделать, чтоб оно просыпалось по ENC_S


на экран вывести индикацию статуса зарядки
и степень зарядки, например в вольтах
и статус подключения USB

пункт меню при выборе которого загоняяем в спячку
при спяке 
* в EN_WAKEUP надо подать низкий уровень
* погасить экран
* погасить неопиксель

при просыпании не обязательно надо подавать высокий EN_WAKEUP может оставаться низким, высокий перед стартом аналоговой части надо подавать
